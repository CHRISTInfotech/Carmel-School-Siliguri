# K.E. Carmel School — Website (Static)

This folder contains a simple, modern, responsive **multi-page** static website for **K.E. Carmel School, Siliguri**, created from the content in `KE School Website Content.pdf`.

## Pages

- `index.html`
- `about.html`
- `management.html`
- `academics.html`
- `admissions.html`
- `facilities.html`
- `gallery.html`
- `contact.html`

## Open locally

- Double-click `index.html`, or
- Run a local server (recommended):
  - `python -m http.server 5500`
  - Then open `http://localhost:5500`

## Update images

- Replace the placeholder SVGs in `assets/img/` with real photos (JPG/PNG/WebP).
- Keep the same filenames to avoid updating HTML, or update the `src` paths in `index.html`.

## Update contact/map

- Edit address/phone/email in `index.html`.
- Replace the Google Maps link query with the exact map pin/link when available.
